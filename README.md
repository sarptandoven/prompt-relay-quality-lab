# prompt relay quality lab

a focused build and update lab for improving the prompt-relay node pack, especially for ltx 2.3 video workflows.

this repo experiments with cleaner prompt structure, stronger scene-to-scene continuity, and better segment timing for prompt-relay. the goal is to make internet-style video prompts easier to test, refine, and reuse without manually rebuilding prompt chains every time.

## focus

- improve prompt-relay behavior for ltx 2.3
- test prompt timing, scene anchors, and transition quality
- refine node behavior for clearer video generation control
- build practical updates that make prompt relay workflows faster and more consistent

## status

work in progress. this repository is for testing, modifying, and improving the prompt-relay node pack.
